# Agent Logs

This directory can store local or committed summaries of agent activity.

Do not commit:

- Raw secrets.
- Private device identifiers.
- Full chat logs containing unrelated personal data.
- API keys or tokens.

Prefer short structured summaries:

```markdown
# Agent Log: Pxx-Jxxx

## Agent
## Start state
## Actions
## Result
## Validation
## Notes
```
